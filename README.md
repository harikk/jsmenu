#JSMenu

This is a jquery plugin for creating menus.
Syntax is very simple.

var menu = $(menu_div_selector).setupMenu(menuObj);

#Options

* _escaped
* _callback
* _dont_hide_on_click
* _no_extra_width
* _no_text
* _icon
* _id
* _disabled
* _no_hover

#API

* setupMenu
* addMenu
* removeMenu
* disable
* enable
